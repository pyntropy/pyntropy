# Определения

- domain - область знаний, тематика. Пример: базы данных, обмен сообщениями.
- modul - модуль. Пример для базы данных: redis, postgres
- step - один шаг модуля. Пример: создание базы данных, создание таблицы.

- title - заголовок домена, модуля или шага модуля.
- overview - краткое описание домена, модуля или шага модуля.
- objectives - список целей изучения домена, модуля или шага модуля.
