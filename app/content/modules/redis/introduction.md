# Введение в Redis

Redis (Remote Dictionary Server) — это хранилище структур данных в памяти, используемое как база данных, кэш и брокер сообщений.

## Основные особенности

- **Хранение в оперативной памяти** — высокая производительность
- **Поддержка структур данных** — строки, списки, хэши, множества
- **Режимы персистентности** — RDB и AOF
- **Репликация** — master-slave архитектура
- **Кластеризация** — автоматическое шардирование

## Примеры использования

1. Кэширование результатов запросов
2. Сессии пользователей
3. Очереди сообщений
4. Системы реального времени
